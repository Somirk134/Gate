# Supported Versions

Gate is currently pre-1.0. Security support focuses on the latest released version and the default
branch until the first stable release.

| Version | Supported | Notes |
| --- | --- | --- |
| `main` | Yes | Active development branch |
| `0.x` latest | Yes | Best-effort security fixes |
| Older `0.x` | No | Upgrade to the latest release |
| `1.x` | Planned | Stable support policy starts at v1 |

## Future Support Policy

After v1, Gate intends to support:

- Latest minor release with security and critical bug fixes.
- Previous minor release for a short migration window.
- Published migration notes for breaking changes.
