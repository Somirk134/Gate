# Design System

## Tokens

| Token | Value | Usage |
| --- | --- | --- |
| Background | `#0B1020` | Primary dark surface |
| Surface | `#111827` | Panels and tool surfaces |
| Border | `#334155` | Separators |
| Primary | `#38BDF8` | Primary action |
| Success | `#A7F3D0` | Healthy state |
| Warning | `#FACC15` | Attention |
| Text | `#F8FAFC` | Primary text |
| Muted | `#CBD5E1` | Secondary text |

## Interaction Rules

- Use icon buttons for common tools.
- Use toggles for binary settings.
- Use segmented controls for modes.
- Use tabs for grouped views.
- Keep cards for repeated items, not page sections.
- Avoid decorative gradients as the primary design language.
