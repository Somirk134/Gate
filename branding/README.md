# Gate Branding

This directory documents brand usage. The primary product logo lives in `assets/logo/logo.svg`.
Open source presentation assets live in `assets/branding`.

## Assets

| Asset | Purpose |
| --- | --- |
| `assets/logo/logo.svg` | Primary product logo |
| `assets/branding/icon.svg` | Small square icon |
| `assets/branding/banner.svg` | README and documentation banner |
| `assets/branding/social-card.svg` | GitHub social preview card |
| `assets/branding/favicon.svg` | Website favicon |
| `assets/branding/open-graph.svg` | Open Graph image |
| `assets/branding/app-icon.svg` | Desktop app icon source |

## Colors

| Token | Hex | Usage |
| --- | --- | --- |
| Gate Ink | `#0B1020` | Background and app shell |
| Gate Sky | `#38BDF8` | Primary action and tunnel path |
| Gate Mint | `#A7F3D0` | Healthy status and secure path |
| Gate Gold | `#FACC15` | Attention and highlights |
| Gate Slate | `#CBD5E1` | Secondary text |

## Usage

- Keep the logo on high-contrast backgrounds.
- Do not stretch, rotate, recolor, or add shadows to the mark.
- Use the social card for GitHub preview images.
- Use the banner for README, documentation, and release pages.
