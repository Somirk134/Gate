# Community Guide

Gate community spaces are for collaborative development, support, design discussion, and release
coordination.

## Channels

| Channel | Use |
| --- | --- |
| Issues | Confirmed bugs, accepted feature requests, documentation defects |
| Discussions | Q&A, ideas, showcase, general support |
| Pull Requests | Concrete changes ready for review |
| Security Advisories | Private vulnerability disclosure |
