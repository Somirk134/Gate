# Discussion Policy

Use GitHub Discussions for conversation that is not yet an actionable issue.

## Categories

| Category | Purpose |
| --- | --- |
| General | Project conversation |
| Ideas | Early proposals |
| Q&A | Questions and answers |
| Showcase | Deployments, demos, and writeups |
| Announcement | Maintainer announcements |
| Support | Help requests that need discussion |

## Guidelines

- Search before posting.
- Include context and version when asking for help.
- Keep proposals focused.
- Convert discussions to issues once work becomes actionable.
