# GitHub Project Template

Use this template when creating the public Gate GitHub Project.

## Views

| View | Purpose |
| --- | --- |
| Board | Todo, In Progress, Review, Done |
| Roadmap | Version-oriented planning |
| Milestones | Release criteria |
| Triage | New issues and discussions |

## Fields

| Field | Type | Values |
| --- | --- | --- |
| Status | Single select | Todo, In Progress, Review, Done, Roadmap, Milestone |
| Priority | Single select | P0, P1, P2, P3 |
| Area | Single select | docs, ci, client, server, runtime, security, docker, website |
| Version | Text | `v1`, `v1.1`, `v1.5`, `v2`, `v3` |
| Risk | Single select | low, medium, high |

## Default Columns

- Todo
- In Progress
- Review
- Done
- Roadmap
- Milestone
