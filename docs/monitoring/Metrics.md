# Metrics

指标体系位于 `crates/domain/src/modules/statistics/metrics` 和 `client/src/monitoring/types`。

## Metric Kinds

- `Counter`: 单调递增计数。
- `Gauge`: 当前值。
- `Histogram`: 分布采样。
- `Summary`: 分位数统计预留。
- `Rate`: 单位时间速率。
- `Average`: 平均值。
- `Peak`: 峰值。
- `Min`: 最小值。
- `Max`: 最大值。
