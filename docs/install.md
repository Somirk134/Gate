# Install

Gate supports source builds, Docker deployment, and desktop development mode.

## Build From Source

```bash
cargo build --workspace --release
```

Server binary:

```text
target/release/gate-server
```

## Install Server Locally

```bash
cargo install --path server
```

## Desktop Client

```bash
cd client
npm install
npm run tauri dev
```

Production desktop packages are generated by the release workflow after signed releases are enabled.

## Documentation Site

```bash
cd website
npm install
npm run dev
```

## Platform Notes

| Platform | Notes |
| --- | --- |
| Linux | Install Tauri system dependencies before building the desktop client |
| macOS | Xcode command line tools are required |
| Windows | Use the Microsoft C++ build tools and WebView2 runtime |
