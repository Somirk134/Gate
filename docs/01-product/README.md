# Product Overview

## Purpose

This directory contains product-level documentation for Gate, including:

- Product vision and strategy
- User personas and use cases
- Feature specifications
- Competitive analysis
- Market positioning

## Contents

- **Vision.md** — Long-term product vision and goals
- **Personas.md** — Target user personas (individuals, teams, enterprises)
- **UseCases.md** — Core use cases and user stories
- **Features.md** — Feature matrix and prioritization
- **Roadmap.md** — Product roadmap aligned with releases
- **CompetitiveAnalysis.md** — Comparison with FRP, ngrok, Tailscale, ZeroTier, etc.

## Why This Design

Product documentation is separated to ensure that feature decisions are
traceable back to user needs. Every technical decision in the architecture
should be justified by a product requirement.

## Extension

Add new product documents as the product evolves. Each major feature should
have a corresponding product spec before implementation begins.
