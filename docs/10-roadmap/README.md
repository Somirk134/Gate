# Roadmap

## Purpose

This directory tracks the project roadmap, milestones, and release planning.

## Contents

- **Vision.md** — Long-term vision (2-3 years)
- **Milestones.md** — Milestone definitions with deliverables
- **Releases.md** — Release schedule and versioning
- **Epics.md** — Epic-level feature tracking
- **Progress.md** — Current progress and status

## Release Plan

| Version | Focus | Target |
|---------|-------|--------|
| 0.1.0 | Project scaffolding and architecture | Q3 2026 |
| 0.2.0 | Core protocol and server tunneling | Q3 2026 |
| 0.3.0 | Client GUI and IPC | Q4 2026 |
| 0.4.0 | Authentication and security | Q4 2026 |
| 0.5.0 | Statistics and monitoring | Q1 2027 |
| 0.6.0 | Deployment and packaging | Q1 2027 |
| 1.0.0 | Production release | Q2 2027 |

## Why This Design

Roadmap documentation ensures transparency about project direction and
helps contributors understand what to work on next.

## Extension

Update the roadmap as priorities shift. Track completed milestones
and add new ones as the project evolves.
