# UI/UX Documentation

## Purpose

UI documentation captures the user interface design, interaction patterns,
and visual design system for the Gate client.

## Contents

- **DesignSystem.md** — Colors, typography, spacing, components
- **Wireframes.md** — Page-level wireframes and layout diagrams
- **Flows.md** — User interaction flows
- **Prototypes.md** — Interactive prototype references
- **I18n.md** — Internationalization strategy

## Key Screens

| Route | Screen | Description |
|-------|--------|-------------|
| `/` | Dashboard | Overview of system status |
| `/tunnels` | Tunnel List | Manage tunnels |
| `/connections` | Connections | Active connections |
| `/settings` | Settings | Application configuration |

## Why This Design

Naive UI provides a comprehensive component library that follows
enterprise design patterns. The design system ensures consistency
across all screens and reduces UI development time.

## Extension

Add new screens following the established patterns. Maintain the design
system documentation as new components are added.
